# Truck Models

Use this directory for truck-variant documentation.

These notes are for whole-vehicle model differences such as:
- dimensions
- wheelbase and track
- drive layout
- sensor placement
- controller and expander presence
- variant-specific hardware assumptions

## Available Models
| Model | Status | Notes |
|---|---|---|
